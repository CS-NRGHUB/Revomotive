# IBS Italia – Preventivi Batterie

App per la generazione di preventivi batterie industriali.

## Come usare

Apri l'app all'indirizzo GitHub Pages del repository.

## Installazione come App (PWA)

**Android (Chrome):**
- Apri il link nel browser
- Tocca il banner "Installa IBS Preventivi" oppure menu → "Aggiungi a schermata Home"

**iPhone/iPad (Safari):**
- Apri il link in Safari
- Tocca Condividi (□↑) → "Aggiungi a schermata Home"

## Funzionalità
- Catalogo 10.301 prodotti con filtri avanzati
- Generazione preventivi PDF
- Storico offerte salvato localmente
- Funzionamento offline
